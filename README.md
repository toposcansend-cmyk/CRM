# CRM Toposcan

Sistema de CRM (Customer Relationship Management) da Toposcan para gerenciamento de propostas e clientes.

## Nova Arquitetura (V3)

- **Cloud Bridge:** Conexão direta entre o site e o Google Sheets via Google Apps Script. 
- **Tempo Real:** Edições e exclusões refletem instantaneamente na planilha.
- **IDs Estáveis:** leads identificados pela linha da planilha, corrigindo erros de seleção.
- **Autenticação:** Sistema de login seguro com preenchimento automático por vendedor.

## Funcionalidades

- 📊 Dashboard com KPIs e métricas de vendas (2024-2026)
- 📋 Ordenação dinâmica de tabelas e filtros avançados
- 📈 Gráfico de Evolução (respeita filtros de ano e vendedor)
- 💰 Calculadora de precificação integrada
- 📅 Sistema de follow-up com priorização inteligente
- 🔍 Insights e oportunidades automáticas
