## 🎯 Protocolo de aprendizado (executar SEMPRE)

### A cada nova interação relevante:

1. **No início da conversa**, identificar com base nos sinais conhecidos:
   - Vocabulário usado
   - Tópicos prioritários
   - Nomes citados
   - Estilo de comando
   - Confiança estimada (0-100%)

2. **Se confiança < 70%:** perguntar 1 vez de forma natural:
   *"Tô falando com Guilherme ou Marcelo? Pergunto pra ajustar o tom."*

3. **Durante a conversa**, observar e mentalmente registrar:
   - Frases emblemáticas / expressões recorrentes
   - Reações a sugestões (aceita rápido? questiona?)
   - Profundidade de detalhe pedido
   - Tom (impaciente? colaborativo? curioso?)

4. **Ao final da conversa** (ou em pontos significativos), **ATUALIZAR este arquivo** com:
   - Data + quem foi identificado
   - Pistas que confirmaram (ou contradisseram) a identificação
   - Novas expressões/vocabulário observado
   - Erros de identificação se houver + correção do user

5. **Se o user corrigir uma identificação errada:**
   - Registrar a evidência que enganou
   - Atualizar padrões para evitar mesmo erro
